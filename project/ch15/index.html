Destructuring Örnekleri

Array Destructuring

JavaScript'te destructuring, bir dizi veya nesnenin elemanlarını ayrı değişkenlere atamak için kullanılan bir sözdizimidir. Bu işlem, kodun daha okunabilir ve daha temiz olmasını sağlar.

Temel Kullanım

Array destructuring kullanımı oldukça basittir. Bir dizi oluşturup, destructuring kullanarak elemanlarını ayrı değişkenlere atayabiliriz. 


const array = [1, 2, 3];
const [first, second, third] = array;
console.log(first);  // 1
console.log(second); // 2
console.log(third);  // 3

Varsayılan Değerler

Dizi destructuring'de varsayılan değerler kullanarak, eğer dizi içindeki elemanlar tanımlanmadıysa bu değerlerin kullanılmasını sağlayabiliriz. 


const array = [1];
const [first, second = 2] = array;
console.log(first);  // 1
console.log(second); // 2

İleri Düzey Kullanım

Dizi destructuring, bir dizi içindeki elemanları birden fazla değişkene atamak için de kullanılabilir. Ayrıca, dizi parçalarının atlanması da mümkündür. 


const array = [1, 2, 3, 4, 5];
const [first, , third] = array; // ikinci eleman atlandı
console.log(first); // 1
console.log(third); // 3

Fonksiyonlarla Kullanım

Destructuring, fonksiyonların argümanlarını daha etkili bir şekilde yönetmek için de kullanılabilir. Bir fonksiyona bir dizi gönderdiğimizde, dizinin elemanlarını direkt olarak destructuring ile alabiliriz. 


function getCoordinates() {
    return [10, 20];
}
const [x, y] = getCoordinates();
console.log(x); // 10
console.log(y); // 20

Sonuç

Array destructuring, JavaScript'te dizileri ve nesneleri daha verimli bir şekilde yönetmek için kullanışlı bir özelliktir. Kodun okunabilirliğini artırır ve değişken atamalarını kolaylaştırır.
