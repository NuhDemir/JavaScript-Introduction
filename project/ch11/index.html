JavaScript Arrow Fonksiyonları

1. Temel Arrow Fonksiyonu

Arrow fonksiyonları, fonksiyon tanımlamayı daha basit bir sözdizimi ile yapmamızı sağlar. Örneğin:

const greet = name => "Merhaba, " + name;

2. Tek Parametre ile Arrow Fonksiyonu

Arrow fonksiyonunun tek parametresi varsa, parantezler isteğe bağlıdır:

const square = num => num * num;

3. Parametresiz Arrow Fonksiyonu

Eğer parametre yoksa, boş parantezler kullanılır:

const sayHello = () => "Merhaba, dünya!";

4. Blok Gövde ile Arrow Fonksiyonu

Birden fazla deyim varsa, süslü parantezler ve return anahtar kelimesi kullanılır:

const multiplyAndLog = (a, b) => {
    const result = a * b;
    return result;
};
