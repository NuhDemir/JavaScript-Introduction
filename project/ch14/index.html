Destructuring Örnekleri

Array Destructuring

Bir array'den elemanları çıkarma örneği.

let numbers = [1, 2, 3];
let [first, second] = numbers;

Destructuring ile Default Değerler

Array'den değer alırken varsayılan değer kullanma.

let numbers = [1];
let [first, second = 2] = numbers;

İç İçe Destructuring

İç içe bir array'den değer çıkarma.

let nestedArray = [1, [2, 3]];
let [first, [second, third]] = nestedArray;

Nesne Destructuring

Nesneden değer çıkarma örneği.

let person = { name: 'Ahmet', age: 30 };
let { name, age } = person;

Nesne Destructuring ile Default Değerler

Nesneden değer alırken varsayılan değer kullanma.

let person = { name: 'Ahmet' };
let { name, age = 25 } = person;

İç İçe Nesne Destructuring

İç içe bir nesneden değer çıkarma.

let person = { name: 'Ahmet', address: { city: 'İstanbul' } };
let { address: { city } } = person;

Fonksiyon Parametrelerinde Destructuring

Fonksiyon parametreleri ile destructuring kullanma.

function greet({ name }) {
    return 'Merhaba ' + name;
}

Fonksiyon Dönüş Değerlerinde Destructuring

Fonksiyondan dönen nesneden değer çıkarma.

function getPerson() {
    return { name: 'Ahmet', age: 30 };
}
let { name } = getPerson();

Rest Operator ile Destructuring

Destructuring ile kalan elemanları alma.

let numbers = [1, 2, 3, 4, 5];
let [first, ...rest] = numbers;

İç İçe Array ve Nesne Destructuring

İç içe bir array ve nesneden değer çıkarma.

let data = [{ name: 'Ahmet' }, { name: 'Ayşe' }];
let [{ name: firstName }, { name: secondName }] = data;
