JavaScript Array Örnekleri

Array Tanımlama ve Eleman Ekleme

Basit bir array oluşturma ve eleman ekleme örneği.

let numbers = [1, 2, 3, 4, 5];

Array Döngü Kullanımı

Array içindeki elemanları döngü ile yazdırma.

let fruits = ["Elma", "Armut", "Muz"];
fruits.forEach((fruit) => console.log(fruit));

Array'de Push ve Pop Metodu

Array'e eleman eklemek ve çıkarmak için push ve pop metodunu kullanma.

let cities = ["İstanbul", "Ankara", "İzmir"];
cities.push("Bursa");
cities.pop();

Array Map Metodu

Map metodu ile array elemanlarının değiştirilmesi.

let numbers = [1, 2, 3, 4, 5];
let squares = numbers.map(x => x * x);

Array Filter Metodu

Filter metodu ile array'den belirli elemanları filtreleme.

let ages = [15, 21, 18, 30, 17];
let adults = ages.filter(age => age >= 18);

Array Reduce Metodu

Reduce metodu ile array elemanlarının toplamını hesaplama.

let numbers = [1, 2, 3, 4, 5];
let sum = numbers.reduce((acc, num) => acc + num, 0);

Array Sort Metodu

Sort metodu ile array elemanlarının sıralanması.

let names = ["Zeynep", "Ali", "Ayşe", "Hakan"];
names.sort();

Array Join Metodu

Join metodu ile array elemanlarının birleştirilmesi.

let words = ["Merhaba", "Dünya"];
let sentence = words.join(" ");

Array Splice Metodu

Splice metodu ile array'e eleman ekleme veya çıkarma.

let colors = ["Kırmızı", "Mavi", "Yeşil"];
colors.splice(1, 1, "Sarı");

Array Concat Metodu

Concat metodu ile array'leri birleştirme.

let array1 = [1, 2, 3];
let array2 = [4, 5, 6];
let combinedArray = array1.concat(array2);
